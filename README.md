## このサイトは何か

**生年月日を入れると、「今のあなたと同じ年齢のときに、世界の作家が書いた名作」を一覧できる無料Webサイト**です。
さらに、あなたの年齢に近い**偉人の言葉**（意味つき）も表示します。

- 公開URL: https://w6d8pzqwyv-ux.github.io/onaidoshi/
- 収益の柱: **本のAmazonアフィリエイト**（「この本を探す」リンク経由で購入されると紹介料が入る）
- コンセプト: 「自分の年齢に重ねるエモさ」

## 仕組み（どうやって動いているか）

1. **本**：あらかじめ `generate-books.ps1` で各年齢の名作を集めて `books-data.js` に保存済み。サイトはそれを読むだけなので**一瞬で表示**される（昔はその場でWikidataを検索していたが、重くてタイムアウトしたため、この「作り置き」方式に変えた）。
2. **あらすじ（作品紹介）**：本ごとに Wikipedia の冒頭文をその場で取得して表示。※物語の筋ではなく「どんな作品か」の概要。
3. **偉人の言葉**：`quotes.js` から、年齢が近い5件の中をランダムに1つ表示。
4. **本の並び**：毎回シャッフルして最大8冊を表示（押すたびに顔ぶれが変わる）。

## よくある更新作業

### ▶ 本を増やす・作り直す
1. `generate-books.ps1` を実行（PowerShellで）
   ```
   powershell -ExecutionPolicy Bypass -File generate-books.ps1
   ```
2. `books-data.js` が新しくなる
3. **`index.html` の `books-data.js?v=3` の数字を1つ上げる**（これをしないと古いデータがキャッシュで残る）
4. Git で push

※有名度のしきい値は `generate-books.ps1` 内の `$ranges`（数字が小さいほど本が増えるが、マイナーな本も混じる）。今は有名度20以上。

### ▶ 偉人の言葉を増やす
1. `quotes.js` に1行追加（形式は既存と同じ：`age`／`person`／`quote`／`source`／`meaning`）
2. **`index.html` の `quotes.js?v=4` の数字を1つ上げる**（キャッシュ対策）
3. push

**鉄のルール：年齢のでっち上げ禁止。** 「何歳のときの言葉か」が出典（著作や演説の年）で確実なものだけ載せる。古代の偉人（孔子など）は年齢が不明なので載せない。

### ▶ 現地で有名な本を手で足す（Amazonで買えるもの限定）
`books-data.js` の該当年齢に1冊追加。形式：`{"t":"書名","a":"著者","y":"出版年","by":"著者の生年","art":"日本語版Wikipediaの記事名"}`
※追加前に、その本がすでにデータに入っていないか（重複）と、Amazon.co.jpで売っているかを確認する。

## 大事なルール・注意

- ✅ `google4cecdc789d3767da.html` は**消さない**（Search Consoleの確認用）
- ✅ 名言は**年齢でっち上げ禁止・出典必須**
- ✅ データを更新したら `?v=数字` を上げる（キャッシュ対策）
- ✅ ヘイト・暴力・危険物などの本は除外（`generate-books.ps1` の `$excludeQid` と `$deny` で対応済み。例：我が闘争・アナーキスト本・マンガ／コミック類）
- ⚠️ 多言語：UIは日本語/英語を自動切替（`?lang=en` で英語表示テスト可）。ただし**本データは今のところ日本語のみ**。英語など他言語の本データを作るのは今後の大きな課題。
